# Pokedex
Pokedex es una app que muestra una lista de 721 pokemones, con sus respectivo detalle por cada pokemon; mostrados en un modal.
La aplicacion se trabajo con el modelo de componentes, los cuales son:
- Header
- PokemonSearch
- PokemonDetails
